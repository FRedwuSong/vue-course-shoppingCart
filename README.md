# vue-course-shoppingCart
vue.js 練習 - 單頁購物車網站

## 功能實作
- 單頁購物車網站
- 透過 Ajax 取得商品列表
- 提供商品搜尋、列表分頁、放入購物車、購物車商品數量增減以及計算總和功能
